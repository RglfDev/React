# 🚀 Mis Proyectos Destacables con React

¡Bienvenido/a a mi repositorio de proyectos! Aquí encontrarás una colección de desarrollos donde he puesto a prueba mis habilidades en el ecosistema de React, enfocándome en crear interfaces dinámicas, navegables y conectadas a servicios externos.

## 🛠️ Tecnologías y Conceptos Aplicados

En estos proyectos he implementado las siguientes funcionalidades clave:

*   **Navegación Dinámica (Routing):** Implementación de **React Router** para gestionar la navegación entre múltiples vistas (Home, Login, Perfil, etc.) sin recargar la página, creando una experiencia de usuario fluida de tipo SPA (Single Page Application).
*   **Conexión con el Backend (Consumo de APIs):** 
    *   Uso de **Axios** para gestionar peticiones HTTP asíncronas y manejo de errores.
    *   Integración con **REST APIs** desarrolladas en **Node.js**.
    *   Conexión con **WebAPI** empresariales desarrolladas en **C# (.NET)**.
*   **Hooks y Estado:** Uso de `useState`, `useEffect` y **Context API** para la gestión del estado global de la aplicación.
*   **Formularios y Validaciones:** Implementación de lógica avanzada para el control de inputs, manejo de eventos y validación de datos de usuario.

## ⚠️ Instrucciones de Inicio Rápido (IMPORTANTE)

Para optimizar el peso del repositorio, la carpeta `node_modules` no ha sido subida. **Es obligatorio instalar las dependencias localmente** antes de intentar correr cualquier proyecto.

Sigue estos pasos para poner en marcha los proyectos:

1.  **Clona el repositorio en tu máquina local:**
    ```bash
    git clone [https://github.com/RglfDev/React.git](https://github.com/RglfDev/React.git)
    ```

2.  **Entra en la carpeta del proyecto específico que deseas ver:**
    ```bash
    cd nombre-del-proyecto
    ```

3.  **Instala todas las librerías necesarias:**
    *(Este comando leerá el archivo `package.json` y descargará automáticamente la carpeta `node_modules`)*:
    ```bash
    npm install
    ```

4.  **Lanza la aplicación en modo desarrollo:**
    ```bash
    npm run dev
    ```
